# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-30

### Added
- 14 Claude Code skills for README auditing
- 3 parallel subagents (first-impression, conversion, competitive)
- 6 benchmark categories: CLI tools, AI/ML, web frameworks, testing, DevOps, library (90 repos total)
- Scoring rubric: Hero (25%), Visuals (20%), Install (15%), Trust (15%), Structure (15%), Differentiation (10%)
- PDF report generation via ReportLab
- Audit history tracking in `.readme-roast/` (git-friendly, in-repo)
- Star-velocity correlation analysis
- Before/after delta comparison
- README rewrite suggestions
- Star killers quick diagnostic
